ROCK Command Variables

Dynamic variables available for use in command responses

User Information

USER
$cmdUserId
The ID of the user who executed the command
USER
$cmdUserName
The username of the command executor (without discriminator)
USER
$cmdUserTag
The full user tag (username#discriminator)
USER
$cmdUserAvatar
URL of the user's avatar image

Bot Information

BOT
$botUserId
The bot's user ID
BOT
$botUserName
The bot's username

Channel Information

CHANNEL
$cmdChannelId
ID of the channel where command was executed
CHANNEL
$cmdChannelName
Name of the channel (returns "DM" for DMs)

Guild Information

GUILD
$cmdGuildId
ID of the guild where command was executed (empty in DMs)
GUILD
$cmdGuildName
Name of the guild (returns "DM" for DMs)

Mentioned Entities

USER
$mentionedUserId
ID of the mentioned user (if any)
USER
$mentionedUserName
Username of the mentioned user (if any)
USER
$mentionedUserTag
Full tag of the mentioned user (if any)
USER
$mentionedUserAvatar
Avatar URL of the mentioned user (if any)
CHANNEL
$mentionedChannelId
ID of the mentioned channel (if any)
CHANNEL
$mentionedChannelName
Name of the mentioned channel (if any)

Functional & Commands components

COMMAND
$args
All arguments passed with the command
COMMAND
$arg[n]
Argument by it's index ex- arg[1]
COMMAND
$random(min-max)
Random Number between minimum and maximum ex- $random(1-100)
COMMAND
$random[option1, option2, option3, ....]
Random word from given list separated by commas ' , ' ex- $random[apple, banana]
COMMAND
$calc[expression]

Time Information

TIME
$time
Current time in HH:MM:SS format
TIME
$date
Current date in YYYY-MM-DD format
TIME
$year
Current year
TIME
$month
Current month (1-12)
TIME
$day
Current day of month (1-31)
TIME
$weekday
Current weekday name (e.g. "Monday")

These variables can be used in command responses to provide dynamic content.
